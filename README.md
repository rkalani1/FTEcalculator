# Faculty FTE Calculator

A single-page tool for estimating and allocating academic faculty full-time-equivalent (FTE) effort across clinical, research, teaching, and administrative activities.

## Usage

Open `index.html` in any modern browser — it is a self-contained static page with no build step or dependencies.

To publish it as a live site, enable **GitHub Pages** for this repository (Settings → Pages → Deploy from branch → `main` / root).
